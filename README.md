# 蓝桥杯嵌入式设计与开发赛道资料库

![STM32](https://img.shields.io/badge/STM32-G431RBT6-blue) ![Board](https://img.shields.io/badge/Board-CT117E--M4-green) ![Library](https://img.shields.io/badge/Library-HAL-orange) ![IDE](https://img.shields.io/badge/IDE-Keil%20MDK%20%2B%20CubeMX-red)

## 📖 项目介绍 (Introduction)

本仓库致力于分享 **蓝桥杯全国软件和信息技术专业人才大赛——嵌入式设计与开发赛道**的相关备赛资料。

仓库内容涵盖了从基础模块驱动到历届省赛、国赛真题的工程源码，以及相关的客观题题库和模拟题资源。所有代码基于官方指定的 **CT117E-M4** 嵌入式竞赛平台（主控芯片 **STM32G431RBT6**）编写，主要使用 **HAL 库** 进行开发。

**开发环境说明：**

* **配置工具**: STM32CubeMX
* **IDE**: Keil MDK (uVision) / VSCode (辅助编辑)
* **调试方式**: DAP-Link (结合官方 LCD 底层 API 进行调试)

希望本仓库的资源能帮助大家更好地掌握嵌入式编程技能，理解赛题逻辑，在比赛中取得优异成绩！

## 🚀 使用指南 (Usage)

### 1. 获取代码

您可以通过以下两种方式获取本仓库代码：

#### 方式1：通过 Git Clone 克隆仓库（推荐，支持后续更新）

打开终端(Bash)，执行命令：

```bash
git clone https://github.com/HelloToday/lq_embedded_platform.git
```

#### 方式 2：直接下载 ZIP 压缩包（快速获取，不支持后续更新）

1. 访问 GitHub 仓库地址：`https://github.com/HelloToday/lq_embedded_platform`
2. 点击仓库页面右上角的 **Code** 按钮
3. 在下拉菜单中选择  **Download ZIP** ，即可获取仓库代码的压缩包

---

### 2. 学习路径与备赛策略 (建议)

对于初学者或备赛同学，建议按照以下步骤使用本仓库资源：

* **第一阶段：模块专项突破**
  进入 `1-模版` 文件夹，先学习并掌握开发板各个外设模块（如 LED, 按键, ADC, PWM 等）的独立使用方法。
* **第二阶段：真题实战演练 (交叉策略)**
  在熟悉模块后，开始完成历年赛题。建议采用以下两种策略 **交叉进行** ，以达到最佳训练效果：
  1. **基于模板法** ：以 `0A基础模版` 为基础，根据赛题要求，像搭积木一样逐步添加所需的外设驱动。这种方式贴近比赛时的快速开发状态。
  2. **从零构建法** ：尝试不依赖现有模板，完全从零开始配置 CubeMX 和编写代码。这种方式能极大锻炼对底层配置的记忆和理解。
* **考前冲刺** ：
  `all-modle` 系列文件夹集成了常用模块，适合作为考前突击复习的参考标准。

### 3. 工程瘦身

由于 Keil 编译会产生大量中间文件（如 `.o`, `.d`, `.crf`, `.dep` 等），导致工程体积庞大。

* **脚本功能** : 根目录下的 `Keilkill.bat` 用于自动删除这些不必要的中间文件。
* **使用时机** : 在将代码上传到 Git 或进行备份压缩 **之前** ，请双击运行此脚本，以显著减小文件体积，方便上传和分享。
